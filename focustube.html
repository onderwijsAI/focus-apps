YouTube API-key

Voer je persoonlijke YouTube Data API v3-key in om video's te kunnen zoeken.

  1. Ga naar console.cloud.google.com en log in met je Google-account.
  2. Klik bovenin op het project-dropdown menu en kies New Project. Geef het een naam (bijv. "FocusTube") en klik Create.
  3. Ga in het menu naar APIs & Services > Library.
  4. Zoek naar YouTube Data API v3 en klik erop.
  5. Klik op Enable.
  6. Ga naar APIs & Services > Credentials.
  7. Klik op + CREATE CREDENTIALS > API key.
  8. Kopieer de key en plak hem hierboven.

Tip: Beperk de key tot alleen de YouTube Data API v3 via Edit API key > Restrict key > API restrictions. Het gratis quotum is 10.000 eenheden per dag (ongeveer 100 zoekopdrachten).

  2. Tik op het Deel-icoon (vierkantje met pijl omhoog) onderaan het scherm.
  3. Scroll omlaag en tik op Zet op beginscherm.
  4. Kies een naam (of laat "FocusTube" staan) en tik op Voeg toe.
  5. De app opent nu als een standalone app zonder Safari-balk.

Let op: gebruik Safari, niet Chrome of een andere browser. Alleen Safari ondersteunt "Zet op beginscherm" op iOS.

Blokkeer youtube.com in Safari zodat je alleen via FocusTube kunt kijken:

  1. Open Instellingen > Schermtijd.
  2. Tik op Beperkingen voor materiaal en privacy en zet het aan.
  3. Tik op Inhoudsbeperkingen (of Store, web, Siri en Game Center).
  4. Tik op Webinhoud.
  5. Kies Beperk niet-geschikt materiaal.
  6. Onder Sta nooit toe, tik Website toevoegen.
  7. Voer in: youtube.com en voeg ook m.youtube.com toe.

Resultaat: YouTube opent niet meer in Safari of andere browsers. FocusTube werkt nog steeds omdat het de embed-player en API gebruikt, niet de website.

Tip: Stel een Schermtijd-toegangscode in die iemand anders beheert, zodat je de blokkade niet makkelijk ongedaan maakt.
